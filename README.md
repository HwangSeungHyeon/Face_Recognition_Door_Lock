  * Anaconda 4.9.2
  * python 3.7.9
  * tensorflow 2.4.1
  * keras 2.4.3
  * cuda 11.0
  * pickle 
  * numpy
  * mtcnn
  * opencv
  * Matplotlib
  * sklearn
