# Face_Recognition_Door_Lock
딥러닝을 기반으로 한 라즈베리파이 얼굴 인식 도어락을 만드는 프로젝트입니다.


## Precondition / 사전 조건
 * Google Colab: 딥러닝 모델 학습에 사용
 * Raspberry pi 4 Model B 8GB: 카메라 및 도어락 제어
 * pi camera: 사용자의 얼굴을 촬영할 카메라


## Function / 기능
  * 사용자 등록
  * 잠금 해제

## How to use / 사용 방법
  * 사용자 등록
  * 잠금 해제

## Built With / 개발에 참여한 사람
 * 황승현 - CNN 모델 생성, 도어락 연결
